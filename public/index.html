📝 Web Scraping

          POST /v1/scrape
          
Scrape any URL and get clean markdown, HTML, or structured JSON via LLM extraction.

        


        

          
🎬 YouTube Transcripts

          POST /v1/youtube/transcript
          
Extract video transcripts, metadata, search within transcripts, or jump to timestamps.

        


        

          
🕷️ Website Crawling

          POST /v1/crawl
          
Crawl entire websites with sitemap discovery, recursive crawling, and adaptive stop conditions.

        


        

          
🔍 Web Search

          POST /v1/search
          
Search the web and optionally scrape results. Supports category filtering.

        


        

          
🤖 AI Interaction

          POST /v1/scrape/:id/interact
          
Browser automation with AI. Click, scroll, fill forms, and execute JavaScript.

        


        

          
📊 Metrics

          GET /metrics
          
Prometheus-compatible metrics for monitoring request rates, latency, cache hits, and more.

        

      

    

  


  
  

    

      

        
Simple Integration

        
Get started in minutes with our SDK or raw HTTP.

      


      

        

          
          
          
        

        

          
# Install SDK
npm install @webcastle/sdk

# Scrape a website
import { WebCastle } from '@webcastle/sdk'

const client = new WebCastle('your-api-key')

const result = await client.scrape('https://example.com', {
  formats: ['markdown', 'json'],
  extract: {
    schema: {
      title: 'string',
      price: 'number'
    }
  }
})

// YouTube Transcript
const transcript = await client.youtube.transcript('https://youtube.com/watch?v=dQw4w9WgXcQ')
